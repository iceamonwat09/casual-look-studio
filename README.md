# Casual Look Studio
**Cardigan Customizer** — Your Style, Your Cardigan

## วิธีเพิ่ม PNG Sticker ใหม่ (ง่ายมาก!)
1. ตัดพื้นหลังออกจากรูปให้เป็น Transparent
2. **Upload ไฟล์ PNG ลงใน folder `stickers/` โดยตรง**
3. เว็บจะโหลด Sticker ใหม่อัตโนมัติ — ไม่ต้องแก้โค้ดใดๆ

รองรับ: .png .jpg .jpeg .webp .gif
