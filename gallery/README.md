# Lookbook Gallery

## วิธีเพิ่มรูป Lookbook บนหน้าแรก (ง่ายมาก!)
1. เตรียมรูปภาพ 4 รูป (แนะนำแนวตั้ง สัดส่วนประมาณ 4:5)
2. **ตั้งชื่อไฟล์ให้ตรงตามนี้ แล้ว Upload ลง folder `gallery/` โดยตรง**
   - `look-1.jpg`
   - `look-2.jpg`
   - `look-3.jpg`
   - `look-4.jpg`
3. หน้าแรกจะแสดง Section "Lookbook" อัตโนมัติ — ไม่ต้องแก้โค้ดใดๆ

หมายเหตุ:
- ถ้ายังไม่มีรูป (หรือรูปหาย) Section จะซ่อนตัวเองอัตโนมัติ เว็บไม่พัง
- อยากเปลี่ยนรูป ก็แค่ Upload ทับชื่อเดิม
